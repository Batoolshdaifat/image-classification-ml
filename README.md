# Image Classification using CNN

## Overview
This project implements a Convolutional Neural Network (CNN) to classify images using the CIFAR-10 dataset.

## Objective
To demonstrate practical skills in deep learning and computer vision using TensorFlow and Keras.

## Dataset
- CIFAR-10 dataset (10 classes of images)

## Model Architecture
- Convolutional Neural Network (CNN)
- MaxPooling layers
- Dense fully connected layers
- Softmax output layer

## Technologies Used
- Python
- TensorFlow / Keras
- Matplotlib

## Results
The model was trained and evaluated on CIFAR-10 dataset with accuracy tracking over epochs.
